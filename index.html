About OGCoin & Open Build

OGCoin is more than a cryptocurrency—it's a funding mechanism that supports open source software development and trains the next generation of developers through Open Build's educational programs.

Developer Education

Open Build provides comprehensive training programs, bootcamps, and mentorship to help aspiring developers learn modern software development skills.

  • Free coding bootcamps
  • Mentorship programs
  • Real-world project experience
  • Career placement assistance

Open Source Funding

Every OGC transaction contributes to a fund that supports open source projects, maintainers, and contributors who build the tools we all depend on.

  • Project maintenance grants
  • Developer stipends
  • Security audit funding
  • Infrastructure support

How OGC Funds Open Source

1

Transaction Fee

A small portion (0.1%) of every OGC transaction goes to the Open Build fund

2

Community Voting

OGC holders vote on which projects and developers receive funding

3

Direct Support

Funds are distributed directly to maintainers, contributors, and educational programs

Why Stellar is Green

No Mining Required

Stellar uses a consensus mechanism that doesn't require energy-intensive mining operations

Efficient Consensus

Stellar Consensus Protocol achieves agreement with minimal computational overhead

Carbon Neutral Hosting

Many Stellar validators run on renewable energy and carbon-neutral hosting providers

Supporting Green Tech

Why OGC Doesn't Use Mining

Unlike Bitcoin or Ethereum, OGC is built on the Stellar network, which uses a different consensus mechanism called the Stellar Consensus Protocol (SCP).
